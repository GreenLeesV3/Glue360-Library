================================================================================
  Glue360 Library — Xbox 360 Recompiler v1.2 — How To Use
================================================================================

This tool recompiles Xbox 360 game ISOs into native Windows executables
using the RexGlue360 SDK. It runs a 7-stage pipeline: extract, init, codegen,
patch, build runtime, build game, deploy.

SUPPORTED GAMES
---------------
  Spider-Man 3 (Title ID 415607E2)       Profile: spiderman3
    - D3D12 backend
    - Save system hooks, cvar patches, FSR upscaling
    - Custom runtime build from SDK source

  Jurassic: The Hunted (Title ID 41560870)  Profile: jurassic_hunted
    - D3D12 backend
    - 60 FPS unlock, RTV render path, texture cache tuning
    - 171 function boundary hints, setjmp/longjmp fix, switch table override
    - Uses prebuilt SDK (no source build needed)

  Spider-Man: Web of Shadows (Title ID 41560815)  Profile: spiderman_wos
    - D3D12 backend, RTV render path
    - 148 function hints for virtual dispatch thunks
    - RTV black screen fix via execute_unclipped_draw_vs_on_cpu
    - Skate3-style performance cvars
    - 60 FPS unlock, shader cache
    - Uses prebuilt SDK (no source build needed)


QUICK START
-----------

  Using the GUI launcher (double-click):
  =======================================
  Just run xbox360-recompiler.exe with no arguments. A file picker wizard
  will guide you through selecting the ISO, output folder, and game profile.

  Using the standalone distribution:
  ==================================
  xbox360-recompiler.exe ^
      --iso  "path\to\game.iso" ^
      --output "path\to\output-folder" ^
      --profile spiderman_wos

  The standalone distribution auto-discovers the bundled SDK at sdk/ and
  bundled tools at tools/. No --sdk flag needed.

  Building from source:
  ====================
  1. Build the app:
     set SDK_ROOT=path\to\RexGlue360Recomp
     build.bat

  2. Create a standalone distribution:
     create_standalone.bat

  3. Run:
     build\xbox360-recompiler.exe ^
         --iso  "path\to\game.iso" ^
         --output "path\to\output" ^
         --sdk   "path\to\RexGlue360Recomp" ^
         --profile spiderman_wos

  For Spider-Man 3 with custom runtime (save system patches):
  ===========================================================
  xbox360-recompiler.exe ^
      --iso  "path\to\Spider-Man 3.iso" ^
      --output "path\to\output" ^
      --sdk  "path\to\RexGlue360Recomp" ^
      --sdk-source "path\to\rexglue-sdk-source" ^
      --profile spiderman3

  For Jurassic: The Hunted (prebuilt SDK, no source needed):
  ==========================================================
  xbox360-recompiler.exe ^
      --iso  "path\to\Jurassic.iso" ^
      --output "path\to\output" ^
      --sdk  "path\to\RexGlue360Recomp" ^
      --profile jurassic_hunted


OUTPUT
------

The pipeline produces a standalone game folder at:
  <output>\standalone\

Containing:
  <game>.exe         — the recompiled native game executable
  rexruntime.dll     — runtime (custom-built if --sdk-source provided)
  <game>.toml        — runtime configuration (cvars, refresh rate, etc.)
  game\              — junction to extracted game data
  user_data\         — saves + shader cache


COMMAND-LINE OPTIONS
--------------------

  --iso <path>          Path to the Xbox 360 ISO (.iso). REQUIRED.
  --output <dir>        Output directory. All artifacts go under here. REQUIRED.
  --sdk <path>          Path to the prebuilt RexGlue360 SDK root.
                        If omitted, auto-discovers bundled SDK at <app>/sdk/.
  --sdk-source <path>   Path to the RexGlue SDK source tree (for custom
                        runtime build). Optional; if omitted, the prebuilt
                        DLL is used and the runtime build stage is skipped.
  --profile <name>      Game profile name (default: spiderman3).
                        Available: spiderman3, jurassic_hunted, spiderman_wos.
  --clean               Wipe state and stage outputs before running.
  --resume              Resume from the last completed stage.
  --check-deps          Run only the dependency checker and print a report.
  --help, -h            Show help text.


PIPELINE STAGES
---------------

  1. iso_extract      Extract Xbox 360 ISO (built-in XDVDFS reader, extract-xiso fallback).
  2. rexglue_init     Create the recompilation project (rexglue init).
                      Injects function boundaries, setjmp/longjmp, switch
                      tables, and invalid instruction hints from the profile.
  3. rexglue_codegen  PPC to C++ code generation.
  4. apply_patches    Copy profile source files + apply runtime patches.
  5. build_runtime    Build custom rexruntime.dll (skipped if no SDK source
                      or profile has requires_sdk_source=false).
  6. build_game       CMake + Ninja + clang-cl game build -> <game>.exe.
                      Applies LTO, static runtime (if SDK from source),
                      ccache (if available), and AVX targeting.
  7. deploy           Assemble portable standalone folder.


TROUBLESHOOTING
---------------

  "RexGlue SDK (prebuilt) [BLOCK] (missing)"
    -> Use the standalone distribution (bundles SDK at sdk/), or pass
       --sdk pointing at your RexGlue360Recomp folder.

  "RTV black screen" (Spider-Man: Web of Shadows)
    -> The game needs execute_unclipped_draw_vs_on_cpu = true to prevent
       EDRAM ownership transfer corruption. This is already in the profile.

  "Call to invalid or unregistered function at 0xXXXXXXXX"
    -> The codegen scanner missed a function. Add the address as a function
       hint in the profile's [functions] section, re-run codegen, rebuild.

  Stage fails, want to debug:
    -> Check <output>\.recomp\logs\recomp.log for detailed error messages.
    -> Use --resume to skip completed stages and re-run only the failing one.


SYSTEM REQUIREMENTS
-------------------

  OS:    Windows 10/11 x64
  CPU:   x86_64 with SSE4.1+ (AMD64 baseline)
  RAM:   8 GB minimum (16 GB recommended for runtime build)
  Disk:  ~15 GB free (ISO + extraction + build artifacts)
  GPU:   D3D12-compatible GPU
